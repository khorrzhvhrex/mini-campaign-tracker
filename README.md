# Heart of the Void Campaign Tracker

A lightweight campaign management tool for Warhammer 40K narrative play.

Tracks:
- Army progression
- Battle history
- Unit performance (Survivors / Marked for Greatness)
- Campaign scoring and state

  ## Features

- Persistent campaign state (localStorage)
- Unit management with notes + datasheets
- Battle log with narrative tracking
- Score tracking system (in development)
- Import / Export for armies and campaigns

  ## Live App

👉 https://yourusername.github.io/hotv-tracker/](https://khorrzhvhrex.github.io/mini-campaign-tracker/

## Usage

1. Enter army names and colors
2. Add units to each army
3. Record battles using the mission log
4. Track survivors and Marked for Greatness units
5. Export data as needed

Data is saved automatically in your browser.

## Data Storage

- Uses browser localStorage
- Data persists between sessions
- Clearing browser data will reset the campaign
